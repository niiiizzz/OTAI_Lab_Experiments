# 📘 Optimization Techniques for AI (MA23434)

This repository contains a collection of lab experiments for the course **Optimization Techniques for AI (MA23434)**. Each experiment is documented with:

- 🎯 Aim  
- ⚙️ Procedure  
- 💻 Code Implementation  
- 📊 Output  
- ✅ Result  

These experiments cover essential optimization methods used in Artificial Intelligence, Operations Research, and Machine Learning.

---

## 📂 List of Experiments

### 1. Linear Programming Problem – Constraint Optimization
- Formulation of linear objective functions  
- Solving using constraints  
- Applications in optimization and resource allocation  

---

### 2. Transportation Problem
- Minimizing transportation cost  
- Methods used:  
  - North-West Corner Method  
  - Least Cost Method  
  - Vogel’s Approximation Method  

---

### 3. Assignment Problem
- Assigning tasks to resources optimally  
- Solved using the Hungarian Method  

---

### 4. Integer Programming Problem – Branch and Bound Method
- Handling integer constraints  
- Implementation of Branch and Bound technique  

---

### 5. Dynamic Programming Problems
Includes:
- 🎒 Knapsack Problem  
- 🔢 Subset Sum Problem  
- 🔗 Longest Common Subsequence (LCS)  

Focus on solving problems using recursion and memoization.

---

### 6. Gradient Descent Methods
- Standard Gradient Descent  
- ⚡ Stochastic Gradient Descent (SGD)  
- Widely used in Machine Learning optimization  

---

### 7. Unconstrained Optimization – Nonlinear Least Squares
- Minimizing nonlinear error functions  
- Applications in curve fitting and regression  

---

### 8. Kuhn-Tucker Conditions – Lagrangian Multiplier Method
- Solving constrained optimization problems  
- Understanding Karush-Kuhn-Tucker (KKT) conditions  

---

### 9. CPM Analysis (Critical Path Method)
- Project scheduling technique  
- Identifying critical and non-critical paths  

---

### 10. PERT Analysis (Program Evaluation Review Technique)
- Probabilistic scheduling method  
- Time estimation using:
  - Optimistic time  
  - Pessimistic time  
  - Most likely time
